 Java String & StringBuilder Programs

This repository contains Java programs to practice String, StringBuilder, and StringBuffer operations.

 Topics Covered

 Concatenate Two Strings
- Using `+` operator
- Using `.concat()` method

 Trim Spaces
- Remove leading and trailing spaces using `trim()`

 Reverse a String
- Reverse a string using a loop

 Count Vowels
- Count number of vowels in a given string

 Check Palindrome
- Check whether a string is a palindrome

 StringBuilder Operations
- `append()` – Add text at the end
- `insert()` – Insert text at specific index
- `replace()` – Replace text between indexes
- `delete()` – Delete characters between indexes
- `length()` – Get length of string

 Concepts Used

- StringBuffer methods
- Loops
- Conditional statements
- Character handling

 Purpose
